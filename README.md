## Laravel Starter

This repo is my simple boilerplate for Laravel apps with Semantic UI. Here is some feature:

-   Laravel 6.2
-   Laravel Default Auth
-   Semantic UI (no sass yet)
-   Yajra DataTables
-   Toastr
-   Font Awesome

Feel free to use it.
